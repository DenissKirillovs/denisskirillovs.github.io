Stylised Environment


Project Type: Personal
Software used: Unreal Engine 5.2
Language Used: Blueprint

  • I've always been a fond of stylised games and environments, so this was a short preview and environment I created to explore it. For this short 1 day project, I used an asset pack to quickly put things together after I did a short sketch of an environment I wanted to create.

    After this project was finished, I am starting to plan out a proper stylised environment designed from the ground up. I want to transfer my skills I've gained using Maya and Substance Painter and start using Blender, Speedtree and other software to help me create environments.

  • I quickly put together a brainstorm Miro page to get the style and look I wanted for this environment. I'm planning on using this moodboard for future stylised environments I make.
